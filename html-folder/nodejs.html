NodeJS

Overview

Node js is an open-source, cross-platform JavaScript runtime environment that allows developers to execute JavaScript code outside the web browser. It was developed by Ryan Dahl and first released in 2009. Node.js utilizes the V8 JavaScript engine, the same engine that powers Google Chrome, to enable server-side and network application development using JavaScript.
